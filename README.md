### 목차  

### 개요 (개발목표)
도서관을 운영하기 위한 가장 기본적인 인프라로써, 이용자 중심의 도서관 웹 사이트를 구축하여 도서관 이용 활성화에 기여하기 위함
### 설계의 주안점
 - 최신 IT 기술을 반영하는 이용자 중심의 맞춤형 서비스 제공  
 - 방대한 컨텐츠 구축과 검색 제공으로 정보 제공자로서의 역할을 강화하며, 도서관 이용자의 지식 경쟁력 강화  
 - 도서관 서비스 수준을 높여 이용자의 접근성 제고
